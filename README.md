# Touhou_stg
北京大学2022春季学期《程序设计实习》大作业.

仿东方的竖版弹幕射击游戏.

- `方向键`移动, 按住`shift`精细移动.
- `z`键射击.
- `esc`暂停, 再次按下`esc`继续.
